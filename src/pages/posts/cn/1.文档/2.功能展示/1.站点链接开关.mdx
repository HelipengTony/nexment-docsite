# 站点链接输入框开关
通过 enableLinkInput 选项配置

<br/>

开启站点链接输入将在评论提交时记录站点链接，在评论列表中点击评论人名称即可访问

### 开启时
```json
{ "enableLinkInput": true }
```
![nexment_siteurl_enabled.png](https://i.loli.net/2020/08/23/37zo6hJZ2Ck5qPi.png)

<br/>

### 关闭时
```json
{ "enableLinkInput": false }
```
![nexment_siteurl_disabled.png](https://i.loli.net/2020/08/23/lVd9aBhKxerUSR7.png)
