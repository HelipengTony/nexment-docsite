# 弹窗列表展示开关
通过 enableReplyListModal 选项配置

<br/>

开启弹窗列表展示将对于评论级别高于 2 的评论隐藏并在点击时以弹窗形式打开。弹窗内包含回复框和评论列表。适用于大量评论/高层级内容。

### 开启时
```json
{ "enableReplyListModal": true }
```
![nexment_modal_enabled.png](https://i.loli.net/2020/08/23/fES85WDor3keC2V.png)
<br/>

### 关闭时
```json
{ "enableReplyListModal": false }
```
![nexment_modal_disabled.png](https://i.loli.net/2020/08/23/AOUWkXom3twKfLu.png)
