# 配置LeanCloud

### 第一步
注册或登录 [LeanCloud](https://leancloud.cn/dashboard/login.html#/signup)

![nexment_leancloud_register.png](https://i.loli.net/2020/08/23/s6YDT8cmH45FPQ3.png)

<br/>

### 第二步
在 [控制面板](https://leancloud.cn/dashboard/applist.html#/apps) 创建一个应用

![nexment_leancloud_app.png](https://i.loli.net/2020/08/23/65MKyk7cWY8nIND.png)

<br/>

### 第三步
前往 你的应用 => **储存** 栏目 => 结构化数据，创建一个名为 "nexment_comments" 的 Class

![nexment_leancloud_class.png](https://i.loli.net/2020/08/23/a9X3DBKxiuQAM5l.png)

<br/>

### 第四步
前往 你的应用 => **设置** 栏目 => 应用 Keys，复制 AppID 和 AppKey

![nexment_leancloud_id.png](https://i.loli.net/2020/08/23/pTo4qFAumUl9sHK.png)

<br/>

### 第五步
配置 REST API 服务器地址。客户端的具体配置方法请参考 [指定客户端的服务器地址](https://leancloud.cn/docs/custom-api-domain-guide.html#hash810845114)

<br/>

### 第六步
前往 你的应用 => **设置** 栏目 => 安全中心 添加你的项目所用域名为 Web 安全域名

![nexment_leancloud_domain.png](https://i.loli.net/2020/08/23/VYd8W5AU9CfE21e.png)
